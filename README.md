## Get Started
### 1. Clone the repository

```shell
git clone https://github.com/farhanizza/application-financial-management.git
cd application-financial-management
```

### 2. Install Package
```shell
npm i
```

### 3. Run the program
Run the program 
```shell
npm start
```
